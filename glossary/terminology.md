# BrandFontsIQ Terminology Glossary

## Overview

This glossary establishes a standardized vocabulary for BrandFontsIQ. Using consistent terminology is essential for clear communication across technical documentation, user interfaces, and customer conversations.

## Typography Terms

### Font
A complete set of characters in a specific size, weight, and style of a typeface.

### Typeface
A design for a set of characters. A typeface may include multiple fonts in various sizes, weights, and styles.

### Font Family
A collection of fonts that share common design traits but vary in weight, width, or style.

### Variable Font
A single font file that contains multiple variations of a typeface, allowing dynamic adjustment of attributes like weight, width, and slant.

### Web Font
A font specifically licensed and formatted for use on websites, typically in formats such as WOFF2, WOFF, TTF, or EOT.

### Font Format
The file format used to store and deliver font data:
- **WOFF2**: Web Open Font Format 2.0, offering 30% better compression than WOFF
- **WOFF**: Web Open Font Format, compressed font format for web use
- **TTF**: TrueType Font, used in both print and web
- **OTF**: OpenType Font, an extension of the TrueType format
- **EOT**: Embedded OpenType, older format primarily for Internet Explorer

### Font Subsetting
The process of creating a reduced version of a font that contains only the specific characters needed, resulting in smaller file sizes.

### Font Loading
The process by which browsers download and render fonts, including various strategies that affect performance.

## Performance Terms

### Font Performance
The impact of typography on technical metrics like page load time, rendering speed, and resource utilization.

### Typography Performance
The broader impact of font selection and implementation on both technical performance and user experience.

### Core Web Vitals
Google's set of specific factors that measure web usability, including:
- **LCP (Largest Contentful Paint)**: Measures loading performance
- **FID (First Input Delay)**: Measures interactivity
- **CLS (Cumulative Layout Shift)**: Measures visual stability

### FOUT (Flash of Unstyled Text)
A visual phenomenon where text initially renders in a fallback font before switching to the web font once loaded.

### FOIT (Flash of Invisible Text)
A visual phenomenon where text is invisible until the web font has loaded.

### Font Display
A CSS property that controls how a font is displayed during loading, with values including auto, block, swap, fallback, and optional.

## Business Impact Terms

### Typography ROI
The financial return on investment from optimizing font selection, implementation, and management.

### Conversion Impact
The effect of typography performance and quality on conversion rates and user engagement.

### Emotional Connection Value
The financial value generated when typography creates an emotional connection with users, based on research showing emotionally connected customers are more valuable.

### Font Optimization Value
The combined business value of improving font performance, consistency, and effectiveness.

### Typography Total Cost of Ownership (TCO)
The complete cost of managing typography assets, including licensing, implementation, maintenance, and opportunity costs.

## Font Management Terms

### Font Inventory
A comprehensive catalog of all font assets used across an organization.

### Font Compliance
Adherence to licensing requirements for font usage across an organization.

### Font Consolidation
The process of reducing redundant font assets to improve efficiency and consistency.

### Font Hosting
The method by which fonts are stored and served:
- **Self-hosted**: Fonts hosted on the organization's own servers
- **CDN-hosted**: Fonts delivered via a content delivery network
- **Third-party hosted**: Fonts served from external services like Google Fonts or Adobe Fonts

### Font Management System
Software that helps organizations manage, deploy, and control font assets.

## BrandFontsIQ-Specific Terms

### Font Strategy Score
A proprietary metric that quantifies the overall effectiveness of an organization's typography strategy.

### Typography Business Impact
The quantifiable effect of typography decisions on key business metrics like conversion, engagement, and brand perception.

### Font Performance Index
A composite score measuring the technical performance aspects of an organization's font implementation.

### Typography Experience Value
The financial value derived from improved user experience due to optimized typography.

### Font ROI Calculator
BrandFontsIQ's tool for calculating the financial return on typography investments and optimizations.

*Note: This glossary will be updated as terminology evolves throughout product development.*
